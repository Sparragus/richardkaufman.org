  • Boardcaster

    Spring 2012

    Tags: Physical Computing, Arduino, C, Ruby on Rails

    Boardcaster is a social network for boardgames. This prototype is an electronic chess board with an integrated chess engine. The board records chess games and broadcasts them live on the Internet through WiFi. Our system also has the unique feature of illuminating valid moves for player when a piece is raised with lights located throughout each square on the board. I worked on building the artificial intelligence/chess engine, interfacing the chess engine with the LED lighting system, and designing the inner organization and outer design of the chessboard.

  • ArduRC

    Fall 2011

    ArduRC was a remote control for an quadcopter. Using an Arduino connected via the ZigBee protocol to a quadcopter, the remote control displayed realtime information such as GPS coordinates, altitude and speed on an embedded screen. Using an accelerometer, ArduRC allowed a user to drive the quadcopter by tilting the remote control. This project has ended and the ArduRC does not exist anymore.
